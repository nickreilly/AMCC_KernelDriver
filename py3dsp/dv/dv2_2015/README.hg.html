HG notes for the DV source 

1.  cloning a copy of DV and pulling updates 

   Clone:
   ---------
   mkdir src
   cd src
   hg clone ssh://s2@iaa//home/s2/src/dv dv

   Updating:
   ---------
   hg pull
   hg update

2.  creating 

   hg init    - create the .hg repository (no files added).
   hg add     - should all all files in the cwd.
   hg commit -u s2 - commit to repository.

3.  commits 

   hg commit  -u tony - to commit changes to repository.

4. hg status 

   hg status            - ? show file present, but not added.
   hg status  --all     - shows all files the status indicators
   
   hg locate            - list files in your repository

5. other commands 

   hg add  path/file       - add a file from the repository

   hg rm  path/file        - remove a file from the repository
