THIS EXTENSION MODULE IS OBSOLETE
Python communicates to the device driver directly now.

to compile: 
type
python setup.py build
in this directory.
(use python2 if python doesn't resolve to 2.2 or later)
this method of compiling modules came with python 2.2

The module allows python access to the PCI card.


to install:
type
python setup.py install
in this directory.

ROOT permissions are probably required.
